BrightSign Bootstrap

Use this page to generate an autorun.brs that you can download to an SD card, put in your BrightSign player and Go!

In order to use this feature, you need to create a manifest file. This is a text file that lists all the files in your site as relative URLs. The advantage of this approach is that, if the network connection slows down or disappears, your sign will continue to work. Once you enter the URL of your manifest, and you have installed the autorun.brs on your BrightSign player, the player will download all the assets mentioned in the manifest, then start playing the initial web page, usually "index.html".
